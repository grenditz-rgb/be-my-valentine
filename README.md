Be My Valentine

A playful Valentine - themed web application built with HTML, CSS and JavaScript. This project explores DOM manipulation, event handling and basic UI animations in a fun and interactive format

-Features
Interactive "Yes" or "No" buttons
Dynamic button behavior
Simple Animations

HTML
CSS
JavaScript
